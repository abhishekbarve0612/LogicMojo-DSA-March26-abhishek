# The Two Pointer Patterns You Should Train This Week

There are roughly 6 important patterns for arrays/strings:
1. Opposite Ends
   - left -> start
   - right -> end
2. Same Direction (Fast / Slow)
3. Pair Sum / Sorted Array
4. Three Pointer / Multi Pointer
5. Partitioning / Rearrangement
6. Palindrome / String comparison

If you cover these, 80% of two-pointer problems become obvious.
